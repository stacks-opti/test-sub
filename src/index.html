BIOS Performance Tweaks !Must be applied manually in your BIOS. Press DEL or F2 at boot. All tweaks are safe and improve performance — nothing is changed automatically.
⚠ Apply these manually in your BIOS. Restart PC and press DEL or F2. Every setting listed is safe and will improve your PC.
🚀
Enable XMP / EXPO (RAM Overclock)
HIGH IMPACT
Most PCs ship with RAM running at 2133MHz instead of its rated 3200–6000MHz. XMP/EXPO is the single biggest free performance gain available.
Steps:
1. Restart → press DEL or F2 to enter BIOS
2. Go to AI Tweaker / OC / Extreme Tweaker / D.O.C.P.
3. Find XMP (Intel) or EXPO (AMD) → set to Profile 1
4. Save & Exit
✓ 100% safe — this is exactly what your RAM was designed for
⚡
Remove CPU Power Limits (PL1/PL2 Unlimited)
HIGH IMPACT
Many motherboards cap CPU power far below its rated TDP. Removing limits lets the CPU sustain full boost speed permanently.
Steps:
1. Enter BIOS → Advanced CPU Config / CPU Power Management
2. Set PL1 / Package Power Limit → 4095W or MAX/Unlimited
3. Set PL2 / Short Power Limit → 4095W or MAX/Unlimited
4. Save & reboot
✓ Safe for stock cooling — only raises temps under sustained heavy load
🌡
Disable CPU C-States (Power Sleep States)
MEDIUM
Stops the CPU from micro-sleeping between frames — eliminates micro-stutters and latency spikes during gameplay.
Steps:
1. Enter BIOS → Advanced / CPU Config / CPU Power Management
2. Find C-States / C-State Control
3. Set to Disabled
4. Save & reboot
✓ CPU uses slightly more idle power but responds instantly to load — ideal for gaming
🔒
Disable Resizable BAR / Smart Access Memory
MEDIUM
ReBAR can cause stuttering in FiveM. Disabling it often improves 1% lows and eliminates sudden frame drops in-game.
Steps:
1. Enter BIOS → Advanced / PCI Subsystem Settings
2. Find Resizable BAR or Above 4G Decoding
3. Set to Disabled
4. Save & reboot
✓ FiveM-specific fix — ReBAR was designed for rasterization workloads, not open-world games
💥
Enable Fast Boot
LOW
Skips POST hardware checks at startup — cuts boot time by 5–15 seconds. Zero gaming performance impact.
Steps:
1. Enter BIOS → Boot / Boot Configuration
2. Set Fast Boot → Enabled
3. Set POST Delay Time → 0 seconds
4. Save & exit
✓ No gaming downside — purely affects startup speed
🔧
Set PCIe to Gen 4 / Gen 3 (Match Your GPU)
MEDIUM
Some boards default the GPU slot to PCIe Gen 1/2, cutting bandwidth in half and bottlenecking GPU transfer speeds.
Steps:
1. Enter BIOS → Advanced / PCI Subsystem / PCIe Config
2. Find PCIe x16 Speed / PCIEX16 Slot
3. Set to Gen 4 (RTX 3000+/RX 6000+) or Gen 3 (older GPUs)
4. Save & reboot
✓ Safe — just matches your GPU's supported PCIe generation

NVIDIA Driver Clean Install

Fully guided — removes all old drivers and installs clean with no bloatware

Why do this? Outdated or corrupted NVIDIA drivers cause FPS drops, crashes, and stutters in FiveM. This process completely removes the old driver and installs a fresh clean version with no NVIDIA spyware or background apps that eat RAM.
Detecting GPU...
Step 1 — Preparation
1
Block Windows from auto-reinstalling old driver
Prevents Windows Update from pushing back the old driver during cleanup
2
Download DDU (Display Driver Uninstaller)
Required — completely wipes all NVIDIA driver files before clean install
Step 2 — Safe Mode Cleanup
3
Set PC to boot into Safe Mode
Safe Mode ensures no GPU drivers are loaded so DDU can fully remove them
4
After reboot: run DDU → "Clean and Restart"
In Safe Mode — open DDU, select NVIDIA GPU, click Clean and Restart. PC reboots to normal mode.
Step 3 — Install New Driver
5
Download NVCleanstall (recommended)
Installs NVIDIA driver while stripping: Telemetry, GeForce Experience, Shield Stream, Shadowplay. Keeps only the display driver + PhysX.
6
Run NVCleanstall → load installer → uncheck bloatware → Install
Open NVCleanstall, point it at the .exe you downloaded.
Uncheck these (bloatware/spyware): GeForce Experience, Telemetry, HD Audio Driver, Shield, Shadowplay, USB-C Driver (unless needed).
Keep these: Display Driver, PhysX.
Click Install and reboot when done.
⚠ Safe Mode boot is set. Save all work and restart your PC now. After DDU runs, restart normally and install driver.
